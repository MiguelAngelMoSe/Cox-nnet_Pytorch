# Cox-nnet_PyTorch 

Este repositorio contiene una implementación moderna de **Cox-nnet**, modernizando la arquitectura original de **Theano** a **PyTorch**, además de múltiples cambios de la propia versión de **Python** al ser más moderna que la utilizada en el modelo **Cox-nnet** original. Este es el repositorio de los autores en el que se encuentra la documentación y el código de **Cox-nnet**: https://github.com/lanagarmire/cox-nnet

## Objetivo del Proyecto
El objetivo es actualizar la librería para que sea compatible con **Python 3.11.15**, aprovechando la aceleración por GPU de PyTorch y facilitando su integración con herramientas modernas de ciencia de datos.
